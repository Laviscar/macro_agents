# v1.6 叙事驱动图 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 把单一主线+累加强度的旧叙事模型,重构成资产/驱动有向图(世界树):节点=资产+因子,边=驱动关系,证据挂边带衰减,据此检测驱动切换并预警。

**Architecture:** 纯函数引擎(strength / driver_shift)+ GraphRepository(config 种子化 + CRUD)+ 叙事管理员重构(consolidation 把证据归因到边、重算、检测切换)+ presenters/Streamlit 新「世界树」页与候选确认面板。严格 TDD,纯函数优先,LLM 用 FakeLLMClient 测。

**Tech Stack:** Python 3.11, pydantic v2, pytest, PyYAML, Streamlit, graphviz(已可用)。

参考 spec:`docs/superpowers/specs/2026-06-01-v1.6-narrative-graph-design.md`(§ 引用即指此)。
